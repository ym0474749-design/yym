# Parental Control App (Advanced Version) | تطبيق المراقبة الأبوية (النسخة المتقدمة)

## [Arabic] وصف المشروع
هذا التطبيق هو أداة متقدمة للمراقبة الأبوية، مصمم لمساعدة الآباء في متابعة نشاط أبنائهم لضمان سلامتهم الرقمية. يتميز التطبيق بالعمل في الخلفية والربط المباشر مع بوت تيليجرام لتلقي التنبيهات وتنفيذ الأوامر عن بُعد.

### المميزات الرئيسية:
*   **التحكم عن بُعد**: عبر بوت تيليجرام (Telegram Bot).
*   **المراقبة الحية**: إمكانية طلب لقطات شاشة أو معلومات الموقع.
*   **إدارة الصلاحيات**: يطلب التطبيق صلاحيات الوصول (Accessibility) لضمان العمل المستمر.
*   **التخفي**: يظهر التطبيق كأداة بحث بسيطة (Google Search) للتمويه.

### طريقة البناء (Build):
هذا المشروع جاهز للبناء باستخدام **Android Studio** أو منصات البناء السحابية مثل **Bitrise** أو **Appcircle**.

## [English] Project Description
This application is an advanced parental control tool designed to help parents monitor their children's digital activity to ensure their safety. The app operates in the background and connects directly to a Telegram Bot to receive alerts and execute remote commands.

### Key Features:
*   **Remote Control**: Via Telegram Bot.
*   **Live Monitoring**: Ability to request screenshots or location information.
*   **Permission Management**: Requests Accessibility services for continuous operation.
*   **Stealth Mode**: Appears as a simple search tool (Google Search) for camouflage.

### How to Build:
This project is ready to be built using **Android Studio** or cloud build platforms like **Bitrise** or **Appcircle**.
